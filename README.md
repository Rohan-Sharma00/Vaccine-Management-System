# Vaccine Management System

A simple web-based application to manage vaccine distribution and patient records.  
It allows users to store, fetch, create, and update patient data efficiently.

## Features
- Add, view, update, and delete patient records  
- Manage vaccine details and availability  
- User-friendly interface for hospitals and medical campaigns  
- Built with **HTML, CSS, PHP, and MySQL**

## Setup
1. Clone the repository  
2. Import the provided SQL file into MySQL  
3. Update database credentials in `config.php`  
4. Run the app on a local server (e.g., XAMPP)

## Usage
Login as an admin or staff to manage vaccine data and patient records.
